    靜態程式庫：SURF 專案概觀
========================================================================

AppWizard 已經為您建立了這個 SURF 程式庫專案。

您的專案中未建立任何原始程式檔。


SURF.vcproj
    這是使用應用程式精靈所產生之 VC++ 專案的主要專案檔。
    它含有產生該檔案之 Visual C++ 的版本資訊，
    以及有關使用應用程式精靈所選取之平台、組態和專案功能的
    資訊。

/////////////////////////////////////////////////////////////////////////////
編譯器和連結器參數已經過修改，可支援 MFC。如果
要在這個專案中使用 MFC ClassWizard，您必須將數個檔案加入至 
專案，包括 "resource.h"、"SURF.rc" 和
包含 resource.h 的 "SURF.h"。如果將 rc 檔加入至靜態程式庫中，您可能會
碰到麻煩，因為在 Dll 或 Exe 中限制只能有
一個 .rc 檔案。這個問題可以藉由將程式庫的 .rc 檔包含到
父專案的 .rc 檔中來克服。
